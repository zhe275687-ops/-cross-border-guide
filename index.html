🛒第一步：选平台

不同平台对应不同市场、不同门槛，个人卖家建议先做一个，跑通再拓。

🇺🇸亚马逊 Amazon
  • 全球最大，流量最高
  • 门槛：营业执照 + 商标（部分类目）
  • 模式：FBA（入仓）或 FBM（自发货）
  • 个人适合：FBM 自发货起步，成本低
  • 开店费：$39.99/月（专业卖家）
需要公司执照
🇷🇺Ozon
  • 俄罗斯最大电商，增速快
  • 门槛较低，个人可开店
  • 中国卖家直接发货或入仓
  • 适合：测试新市场，竞争比亚马逊小
个人可注册
🌏Shopee / Lazada
  • 东南亚 + 台湾市场
  • 客单价低但单量大
  • Shopee 跨境店门槛不高
  • 适合：低价走量产品
🎵TikTok Shop
  • 短视频+直播带货，流量红利
  • 美区/东南亚/英国
  • 内容能力要求高
  • 适合：有内容创作能力的卖家
需要内容运营
🇧🇷美客多 Mercado Libre
  • 拉美最大电商
  • 墨西哥、巴西最热
  • FBM自发货为主
  • 适合：拓展新兴市场
🛍️Temu（半托管）
  • 拼多多海外版，增长极猛
  • 全托管:平台包揽物流售后
  • 半托管:卖家自己发货
  • 利润薄但量大，适合工厂型卖家
利润薄需走量
💡 个人新手最推荐的路径：先做亚马逊FBM（练手）→ 同时开Ozon（拓市场）→ 有心力再做TikTok Shop

🌐网络环境 — 跨境生意的"地基"

跨境=翻墙。网络不稳，账号被封，前面白干。

1.1 科学上网（翻墙）

⚡机场/SSR 节点
  • 按月付费，几块到几十块/月
  • 节点遍布全球，切换方便
  • 推荐：自己搜"稳定机场"，看评价
  • 常见协议：SS / V2Ray / Trojan / Hysteria2
¥10-50/月
🖥️自建 VPS 节点
  • 买海外 VPS 自己搭
  • 独享IP，更稳定、更安全
  • 需要一点技术基础
  • VPS：搬瓦工 / Vultr / DigitalOcean
$5-20/月

1.2 防关联 — 多店铺必修

⚠️ 平台会检测浏览器指纹、IP、Cookies，同一台电脑登多个同平台账号会被关联封号！
🧩指纹浏览器
  • 每个店铺一个独立浏览器环境
  • 不同指纹（UA/时区/语言/分辨率）
  • 推荐：AdsPower / Multilogin / 候鸟
  • 免费版支持少量环境
免费起 / 高级版 ¥99+/月
☁️远程 VPS（云桌面）
  • 一台海外 VPS 就是一个独立电脑
  • 纯国外IP，防关联效果最好
  • 淘宝搜"跨境VPS"或自购
  • 智赢ERP内置VPS绑定
¥50-200/月
📱手机 + 流量卡
  • 最原始也最稳的方法
  • 一机一卡一号，永不关联
  • 买二手手机 + 独立流量卡
手机 ¥300-800 + 流量卡 ¥19/月

4.2 选品方法和标准

📐新手选品标准
  • 售价 $15-50（太低没利润，太高没胆量）
  • 重量 < 1kg（运费低）
  • 无电子/液体/粉末（物流限制少）
  • 月搜索量 > 5000
  • 头部竞品评论 < 500（还有机会）
  • 毛利率 > 30%
🎯选品思路
  • 利基市场：大卖看不上的小众品类
  • 改良款：看差评，改掉竞品的缺点
  • 视觉款：换个颜色/图案/包装就变新品
  • 套装组合：把几样东西打包卖
  • 节日/季节款：圣诞、万圣、Prime Day
  • 跟卖自有品牌：看工厂有什么就卖什么
🚫 新手避坑：不碰侵权（大牌Logo）、不碰认证品（FDA/CE/CPC费钱费时）、不做易碎品（售后噩梦）

🚚物流 — 货怎么出国

自发货主要物流方式

方式时效费用适合
国际快递（DHL/UPS/FedEx）3-7天高高价值/急单
专线物流（云途/燕文/递四方）7-15天中最常用！性价比高
邮政小包（EUB/中国邮政）15-30天低低客单价轻小件
海外仓（FBA / 第三方仓）1-3天（本地发货）头程+仓储费稳定出单的爆品

找货代

💡 加5-10个货代的微信比价 → 先发小包测试时效和服务 → 选2-3家长期合作
💡 货代一般在深圳/广州/义乌，离货源近
💡 问清楚：计费方式（体积重还是实重）、时效承诺、丢件赔付、能不能走带电/液体
